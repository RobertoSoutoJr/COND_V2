# COND2
COND é um projeto acadêmico em desenvolvimento que visa criar uma aplicação web e mobile, utilizando React e Spring Boot, para gerenciar condicionais de roupas em lojas. O sistema facilitará o controle de peças em consignação, oferecendo organização e eficiência.
